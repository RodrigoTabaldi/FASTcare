# Imagens para Balões de Dica

Este diretório contém as imagens infográficas que aparecem nos balões de dica educativos.

## Imagens Necessárias

1. **mudanca-posicao.png** - Infográfico de posições e exercícios
   - Exibido no balão: "Mudança de posição"
   - Relacionado ao sinal: "Prevenir úlceras e aspiração"
   - Etapa: 7 (Demência grave)
   - Dimensões recomendadas: 320x150px ou proporção similar

2. **rotina-visual.png** - Infográfico da rotina visual do dia
   - Exibido no balão: "Rotina visual do dia"
   - Relacionado ao sinal: "Vocabulário muito limitado ou ausente"
   - Etapa: 7 (Demência grave)
   - Dimensões recomendadas: 320x150px ou proporção similar

## Como Usar

Coloque as imagens PNG neste diretório e elas serão automaticamente carregadas nos balões de dica correspondentes no componente Tip.tsx.

Se as imagens não forem encontradas, o componente renderizará ilustrações SVG como fallback.
